Design System

Avatar

Avatars are used to represent a person or object to give them a visual identity. They are mostly found in circular shapes but we will provide you with other shapes too.

Images Avatar

If you have a img to use then you can try this image Avatar. Use the class avatar and an image to create a image avatar.

Initials Avatar

If you don't have an image to set then you can also add your initials using a class called avatar--initial.

SK

Sizes Avatar

If you are not satisfied with the size of avatar then you can use our custom classes to get desired results. Use small, large, medium classes for custom size.

Shapes Avatar

You can also shape the avatars using classes --square, --rounded, --circle etc.

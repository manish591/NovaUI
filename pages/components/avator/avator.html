Design System

Avator

Avatars are used to represent a person or object to give them a visual identity. They are mostly found in circular shapes but we will provide you with other shapes too.

Images Avator

If you have a img to use then you can try this image Avator. Use the class avator and an image to create a image avator.

Initials Avator

If you don't have an image to set then you can also add your initials using a class called avator--initial.

SK

SK

Sizes Avator

If you are not satisfied with the size of avator then you can use our custom classes to get desired results. Use small, large, medium classes for custom size.

Shapes Avator

You can also shape the avators using classes --square, --rounded, --circle etc.

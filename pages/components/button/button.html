Design System

Button

Buttons are everywhere on the web. They represent actions that user can take. We can see buttons throughout the we like in cards, in modals etc.

Contained Buttons

This is the most basic type of button you can see on the web. Use the class button ans button--contained.

Outlined Buttons

Outlined button are not filled but have a border on them. Use button--outlined-primary class to create a outlined button.

Text Buttons

Sometimes you can see a text button. Use class btn--text-primary.

Link Buttons

Link can lead you to a diffrent page while button denotes action. You can style a link as a button using our predefined classes. Always use anchor tags for link button.

Icon Buttons

Icon button are very common that you can see in ecommerce websites.

Floating Action Buttons

Normally found in circular shapes with a icon in middle. It perform primary task in the page.

Design System

Introduction to Nova

Nova is a collection of CSS modules, layouts and components to kick start any web project. The library is open source and maintained by the community. Contributing to it will help you build beautiful responsive sites in no time. This is a collection of the most common design patterns, reusable components and best practices to use in your next project. It aims to increase productivity by allowing you to focus on designing beautiful interfaces, instead of wasting time thinking how to build it.
